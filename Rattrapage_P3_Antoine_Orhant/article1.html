Notre premier projet a l’école : le personal branding

Pour entrer dans le vif du sujet, nous avons dès la rentrée appris à se vendre en créant une page internet, on appelle cela le “Personal Branding”. De plus, nous devions non seulement écrire les texte, mais aussi écrire le code en HTML et CSS tout en utilisant Github. Chose que nous ne connaissions pas pour la plupart d’entre nous.

Pour travailler de façon plus éfficace, nous avons découvert la platforme Github qui permet de sauvegarder plus facilement ses projets. Github permet aussi de partager ses projets de façon publique ainsi que de garder trace de tout changement pour éviter par éxemple de sauvegarder une modification de code qui "casse" qui marchait auparavant.
